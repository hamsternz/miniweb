Welcome to Miniweb

Miniweb is a small web server.

It is written in C, and intended to be used in embedded projects, like on the Raspberry Pi.

It is able to concurrently handle multiple concurrent connections, but is non-blocking so you can carry on running other code.

The project home page is at https://github.com/hamsternz/miniweb.
